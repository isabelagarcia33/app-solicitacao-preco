# 📊 App de Solicitação de Preços

Projeto de automação de solicitações de preços com foco em eficiência operacional, padronização de processos e integração de dados.

## 🚀 Contexto

O processo de solicitações era manual, realizado por e-mail, sem padronização e com alto volume operacional.

## 🎯 Objetivo

Automatizar o fluxo de solicitações de preços, reduzindo retrabalho e melhorando a rastreabilidade das informações.

## 🛠️ Tecnologias

- AppSheet (no-code)
- Google Apps Script (JavaScript)
- Google Sheets
- Excel

## ⚙️ Como funciona

1. Usuário envia solicitação via formulário (AppSheet)
2. Dados são armazenados em Google Sheets
3. Script automatiza:
   - Leitura do tipo de solicitação
   - Extração de dados de arquivos enviados
   - Inserção em base consolidada
4. Dados ficam prontos para análise e importação
5. Pós análise, o script faz envio automático de e-mail contendo o status da solicitação para os solicitantes

## 📈 Resultados

- Redução de processos manuais
- Aumento de eficiência operacional
- Padronização de solicitações
- Melhoria no tempo de resposta (SLA)

## 💡 Aprendizados

- Automação de processos com Apps Script
- Estruturação de dados para análise
- Integração entre ferramentas no-code e programação
